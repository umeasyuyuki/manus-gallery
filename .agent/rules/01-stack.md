# 技術スタック (Technology Stack)

## ランタイム

- **Node.js**: Latest LTS (v20+)

## パッケージマネージャー

- **npm**: 使用中 (Next.js init fallback)
- 代替: pnpm, yarn

## フレームワーク / ライブラリ

- **Next.js**: Main Framework (Static Export preferred for MVP if possible, or Vercel default)
- **React**: UI Library
- **Tailwind CSS**: Styling Framework

## 言語

- **TypeScript**: メイン開発言語

## データ管理 / CMS

- **JSON**: Static data for MVP (projects.json)
- **Google Forms**: Project submission (External)

## 開発ツール

- **ESLint**: Linting
- **Prettier**: Formatting

## ホスティング / デプロイ

- **Vercel**: Preferred hosting platform
- **GitHub Pages**: Alternative

## ビルドコマンド

```bash
pnpm build
pnpm dev
pnpm lint
```
